 Smart Food Delivery Web Application

A full-stack food ordering web application built using Python Flask that allows users to browse restaurants, view food menus, add items to cart, and place orders with a checkout system including delivery location and payment options (Cash/UPI).

---

 Features

-  Restaurant listing system  
-  Dynamic food menu for each restaurant  
-  Add to Cart functionality  
-  Remove items from cart  
-  Automatic total price calculation  
-  Checkout system with delivery details  
- Payment option simulation (Cash / UPI)  
-  Order placement confirmation page  

Tech Stack

- Python  
- Flask (Backend Framework)  
- HTML5  
- CSS3  
- Jinja2 (Template Engine)
